## :warning: I'm moving away to Codeberg :warning:

I've decided to leave Github and move to the free software alternative [Codeberg](https://codeberg.org/).

I've always been in favor of the free software movement and I've tried to follow its philosophy in all the software I've released.

I've understood that therefore it doesn't make sense to host my free software projects in a proprietary forge like Github.

Given that Github is and will be for a long time the de-facto standard in the software industry for hosting and managing software projects, I will still leave all the work I've done until now here.

I will be archiving all my work and slowly migrate to Codeberg, so for anyone interested, check my most up to date work in my [Codeberg profile](https://codeberg.org/JasterV?tab=overview)
